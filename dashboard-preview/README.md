# Dashboard Preview

Interactive Power BI dashboard built for the ETF vs Traditional Banking Investment Analysis project.

## Pages

**Page 1 — Executive Overview**
High-level comparison of $10,000 invested in SMH, VTI, and CD over 20 years. Includes compound growth line chart, annual returns bar chart, KPI cards, and year slicer.

**Page 2 — Instrument Deep Dive**
Per-instrument analysis with interactive slicer (VTI / SMH / CD). Shows capital growth, yearly gain/loss, best and worst years, and full data table.

**Page 3 — Risk vs Return**
Risk-adjusted performance analysis. Includes scatter plot (risk vs return), volatility metrics, max drawdown cards, and positive vs negative years chart.
